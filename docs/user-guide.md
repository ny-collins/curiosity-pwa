# User Guide

Welcome to Curiosity, your personal knowledge management companion! This guide will help you get started with all the features and make the most of your journaling experience.

## Getting Started

### Installation

#### Web Browser
1. Visit [https://curiosity-pwa.web.app](https://curiosity-pwa.web.app)
2. Click the install prompt or browser menu ‚Üí "Install Curiosity"
3. The app will be added to your home screen

#### Mobile Devices
- **iOS**: Tap the share button ‚Üí "Add to Home Screen"
- **Android**: Tap the menu ‚Üí "Add to Home Screen" or "Install App"

### First Time Setup

1. **Welcome Screen**: Choose your preferred theme and language
2. **Account Setup**: Sign in with Google for data synchronization
3. **Profile Setup**: Add your name and profile picture
4. **Security Setup**: Set up a PIN for app security
5. **Biometric Setup**: Enable fingerprint/face unlock (optional)

## Core Features

### üìù Creating Entries

#### Journal Entries
1. Tap the **+** button or "New Journal" on the dashboard
2. Choose "Journal" type
3. Write your thoughts with rich text formatting
4. Add tags for organization
5. Auto-save ensures your work is never lost

#### Notes
1. Select "New Note" from the dashboard
2. Use Markdown for formatting:
   - `# Heading` for headings
   - `**bold**` for bold text
   - `*italic*` for italic text
   - `[link](url)` for links
   - `` `code` `` for inline code

#### Tasks
1. Choose "New Task" type
2. Add task description
3. Set due dates and priorities
4. Mark tasks as complete

### üìÖ Calendar View

- **Navigate**: Swipe or click to change months
- **View Entries**: Dates with entries are highlighted
- **Quick Access**: Tap any date to see entries for that day
- **Entry Dots**: Colored dots indicate entry types

### üéØ Goals & Tasks

#### Creating Goals
1. Go to Goals section
2. Tap "New Goal"
3. Set title and description
4. Add tasks to break down your goal
5. Track progress automatically

#### Managing Tasks
- **Add Tasks**: Tap the + button in any goal
- **Complete Tasks**: Check off completed items
- **Reorder**: Drag tasks to prioritize
- **Delete**: Swipe left on tasks to remove

### üîí Secure Vault

#### Setting Up the Vault
1. Set an app PIN in Settings ‚Üí Security
2. Enable biometric authentication (recommended)
3. Access vault from the main menu

#### Adding Vault Items
- **Passwords**: Store website credentials securely
- **Contacts**: Keep sensitive contact information
- **Notes**: Encrypted private notes

#### Security Features
- **Client-Side Encryption**: Data encrypted before leaving your device
- **PIN Protection**: Required to access vault
- **Biometric Unlock**: Fingerprint/face recognition
- **Auto-Lock**: Vault locks after inactivity

### üîî Reminders

#### Setting Reminders
1. Go to Reminders section
2. Tap "Add Reminder"
3. Set title, date, and time
4. Choose notification preferences

#### Managing Reminders
- **Upcoming**: View scheduled reminders
- **Past**: Review completed reminders
- **Notifications**: Get alerted at the right time

## Troubleshooting

### Common Issues

#### App Won't Load
- **Clear Cache**: Browser settings ‚Üí Clear site data
- **Update Browser**: Ensure latest version
- **Check Network**: Verify internet connection

#### Entries Not Saving
- **Check Connection**: Ensure online status
- **Auto-save**: Wait for auto-save indicator
- **Manual Save**: Use Ctrl/Cmd + S

#### Biometric Login Fails
- **Re-enroll**: Settings ‚Üí Security ‚Üí Reset biometrics
- **Browser Support**: Check WebAuthn compatibility
- **Device Settings**: Verify biometric setup

#### Sync Issues
